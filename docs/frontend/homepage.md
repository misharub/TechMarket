# Главная страница TechMarket

## Назначение

Главная страница работает как коммерческая витрина интернет-магазина техники. Она не использует mock-данные: категории и товары загружаются из backend API.

## Структура

- Hero-блок с основным оффером, CTA в каталог и быстрым переходом в первый раздел.
- Smart rail: компактные предложения, включая товар со скидкой и быстрые ссылки на категории.
- Блок преимуществ: доставка, самовывоз, оплата, гарантия.
- Быстрые категории из `/categories/tree`.
- Товарные секции:
  - «Новинки каталога»;
  - «Скидки недели»;
  - «Премиальная техника».

## API-запросы

- `GET /categories/tree` — дерево категорий для быстрых переходов.
- `GET /products?limit=8&sort=newest&inStock=true` — новинки.
- `GET /products?limit=8&sort=priceDesc&inStock=true` — дорогие и флагманские товары.
- `GET /products?limit=24&sort=newest&inStock=true` — источник для скидок; товары со `oldPrice` фильтруются на клиенте.

## Состояния

- Loading: skeleton-плашки без спиннеров.
- Error: понятное сообщение о недоступном backend.
- Empty: сообщение, что в подборке пока нет товаров.
- Изображения: если `/uploads/products/...` недоступен, показывается аккуратный placeholder с брендом или категорией.
