# Changelog TechMarket

## 2026-05-11

- Главная страница переведена в формат API-driven витрины: hero, smart rail, быстрые категории, товарные секции, skeleton/error/empty состояния.
- Добавлена общая карточка товара для главной и каталога.
- Добавлен каталог v1: маршруты `/catalog` и `/catalog/:categorySlug`, фильтры, сортировка, поиск, пагинация и URL-параметры.
- Добавлен frontend API для брендов через `GET /brands`.
- `GET /products` расширен параметром `categorySlug`; выбранная категория включает дочерние категории.
- Каталог перестроен по сценарию DNS/Komputronik: `/catalog` показывает крупные корневые разделы, корневой раздел показывает только подкатегории, а фильтры и товары появляются на странице конкретной подкатегории.
- Добавлена документация по главной странице, каталогу и API товаров.
- Добавлена инфраструктура служебных страниц для ссылок header/footer: контакты, магазины, сравнение, корзина, вход, страницы компании и помощи.
- Служебные страницы разнесены на отдельные page-компоненты, чтобы каждую страницу можно было редактировать независимо.
