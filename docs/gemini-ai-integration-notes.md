# Интеграция Gemini AI для сравнения товаров

Этот файл дополняет основную памятку по серверу и описывает реальную AI-интеграцию, добавленную после подготовки Google OAuth.

## Назначение

В проекте уже был локальный demo AI-like анализ сравнения товаров. Теперь сервер умеет подключать внешний AI-сервис Google Gemini и использовать его для формирования более естественного заключения по таблице характеристик.

## Переменные окружения

```env
AI_PROVIDER="gemini"
GEMINI_API_KEY=""
GEMINI_MODEL="gemini-2.5-flash"
```

`GEMINI_API_KEY` хранится только в `.env` на сервере и не должен попадать на GitHub.

## Как работает сравнение с AI

1. Frontend отправляет 2-3 `productIds` на `POST /api/products/compare`.
2. Сервер проверяет, что товары существуют, активны и относятся к одной категории.
3. Сервер строит таблицу характеристик по шаблонам категории.
4. Сервер считает лучшие значения для подсветки в таблице.
5. Сервер формирует резервный локальный анализ.
6. Если `AI_PROVIDER=gemini` и `GEMINI_API_KEY` задан, сервер отправляет структурированные данные в Gemini.
7. Если Gemini отвечает успешно, поле `aiSummary` содержит реальный AI-ответ.
8. Если Gemini недоступен, сервер возвращает локальный fallback.

## Формат ответа

```json
{
  "aiSummary": "Для покупателя наиболее сбалансированным вариантом выглядит...",
  "aiSummaryMeta": {
    "provider": "gemini",
    "isFallback": false
  }
}
```

Если внешний сервис недоступен:

```json
{
  "aiSummary": "Demo AI-анализ сформирован сервером по правилам...",
  "aiSummaryMeta": {
    "provider": "local-rules",
    "isFallback": true,
    "fallbackReason": "Gemini API returned 429"
  }
}
```

## Что можно написать в дипломе

В системе реализован модуль AI-анализа сравнения товаров. При сравнении техники сервер формирует таблицу характеристик, выделяет лучшие параметры и передает структурированные данные во внешний LLM-сервис Google Gemini. Полученное заключение помогает пользователю выбрать наиболее подходящий товар. Для повышения надежности предусмотрен локальный fallback: если внешний AI-сервис недоступен или исчерпан лимит запросов, сервер формирует резервное заключение на основе правил.
